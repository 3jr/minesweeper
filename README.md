# minesweeper
Solves an ever expanding game of minesweeper

I dug out this 3-year-old project because I wanted to try to port something to .NET Core.
I literally just copied the sources and it worked.
I also added command line parsing and a few modes that didn't exist before.

# usage

You must have .NET Core installed (https://www.microsoft.com/net/download/core).

Example:
